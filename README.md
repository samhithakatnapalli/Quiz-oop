# Percy Jackson Quiz (OOP)

  A fun, simple, command-line quiz game inspired by the Percy Jackson & the Olympians universe
  Built in Python using Object-Oriented Programming (OOP) concepts to keep the code clean, and readable.

## Features:
  - A quiz based on Percy Jackson trivia with right answer for every question displayed after.
  - Score tracking as the quiz progresses.
  - Clear separation of concerns using classes.
  - Easy to extend with more questions.

## Project structure:
  ```
  Quiz-oop/

  ├── main.py              # Entry point of the quiz
  ├── question_model.py   # Question class
  ├── quiz_brain.py       # Quiz logic and flow
  ├── data.py              # Question data
  ```
